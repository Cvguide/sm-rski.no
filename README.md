# Smøreguide - Skismøring og Løyper

Din digitale skismøringsassistent med værmelding og løypeforslag for hele Norge.

## Funksjoner
- Værmeldinger fra Meteorologisk institutt
- Smøringsanbefalinger basert på Swix retningslinjer
- Oversikt over skiløyper i Norge
- Løypeinformasjon fra UT.no og andre kilder

## Installasjon

```bash
npm install
npm start
```

## Deploy til Vercel

1. Push til GitHub
2. Koble GitHub-repo til Vercel
3. Deploy automatisk
